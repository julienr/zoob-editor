BETA VERSION

The Zoob level editor is currently in BETA version. This means that it might still contains bugs, so bear with us if some things don't work as expected.

To report bugs, please send a mail to zoobdev@gmail.com describing the situation as well as your android version

General help

This application allows you to create level series for Zoob. A level serie contains multiple levels and the player has to go through the various levels in a given order.

Once you have created a serie, you can upload the serie to the web, where it will be shared with the community.

You can also download and play other player's series by using the Community tab.
